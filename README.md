# Binance ZK Distribution Program Checker

- **Python 3.10**

```
git clone https://github.com/Eazer1/Binance_ZK_Distribution_Program
```
```
cd Binance_ZK_Distribution_Program
```
**Впишите адреса ваших кошельков в wallets.txt**
```
python main.py
```
